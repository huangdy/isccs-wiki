# facility-getUsingAgency
### Description  
This API returns the list of name/id for all the using agency/bureau from the facility table.
       
### Params

### Example Request  
http://localhost:8080/isccs/api/facility/getUsingAgency
```
### Example Response  
```javascript
[
    {
        name: "House of Representatives",
        id: "0005"
    },
    {
        name: "Senate",
        id: "0019"
    },
    .
    .
    .
]
```
